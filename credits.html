Ice-Spear - Credits

Ice-Spear is an open source editor (GNU GPLv3).
It's free, and doesn't pursue any financial goals, neither with fees nor with donations.
This Project also does NOT contain any files from games that it may be compatible with.
You need to have your own, legally owned games or copies, containing those files.
